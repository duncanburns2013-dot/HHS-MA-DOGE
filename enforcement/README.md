# ENFORCEMENT LAYER

This folder collects external enforcement and oversight activity that corroborates or extends the patterns documented in the rest of this repo. Nothing here is original investigation — it is a curated index of government-issued reports, indictments, and recoveries from federal and Massachusetts authorities, with pointers back to the entities and patterns this repo already analyzes.

## Why this folder exists

The core repo names specific entities, controllers, and dollar amounts based on T-MSIS, CTHRU, OCPF, IRS 990, and lobbying records. This enforcement layer answers two questions reviewers will ask:

1. **Is anyone in the government already looking at this?** (Yes — DOJ stood up a New England Strike Force specifically for MA healthcare fraud, and the State Auditor's Bureau of Special Investigations identified ~$12M in MassHealth/DTA/EEC fraud in FY25 alone.)
2. **Where do these patterns end up when prosecutors do touch them?** (Recent MA AG indictments show the same structural patterns — single controller, multiple LLCs, identical billing codes — landing as money-laundering and Medicaid-fraud charges.)

## Source policy

Every claim in this folder cites a primary government URL (mass.gov, oig.hhs.gov, justice.gov) or a recognized news outlet linking to one. No social media, no blogs, no reddit. If a link rots, replace with the next-best primary source — never delete a citation.
