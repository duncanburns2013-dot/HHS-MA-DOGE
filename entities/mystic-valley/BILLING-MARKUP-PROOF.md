## THE TWO-LAYER MARKUP

### LAYER 1: State → Mystic Valley (the admin cut)

```
State pays MVES (CTHRU FY2025):     $102,077,368
MVES passes to subcontractors (990): $87,092,793
                                     ───────────
MVES KEEPS:                          $33,282,342  (27.3%)
```

What MVES keeps pays for:
- 6 executives making $1,234,448 combined (Schedule J)
- Staff salaries of $18,757,020
- Rent $1,219,964, meals program $4,603,382, client transport $1,374,372
- 13 employees earned >$100K

### LAYER 2: Subcontracted Agency → Home Health Aide

```
MassHealth G0156 rate:               $40.72/hr
Aide actual wage (BLS MA mean):     ~$18.50/hr
                                     ──────────
Agency overhead spread:             ~$22.22/hr  (54.6%)
```

## THE COMPLETE CHAIN

```
TAXPAYER DOLLAR
      │
      ▼
STATE OF MASSACHUSETTS (MassHealth/Elder Affairs)
      │ pays $40.72/hr per G0156 unit
      ▼
MYSTIC VALLEY ELDER SERVICES (nonprofit middleman)
      │ keeps 27.3% ($33.3M)
      │   → CEO: $299,064
      │   → CFO: $250,325
      │   → 13 staff earn >$100K
      ▼
FOR-PROFIT HOME CARE LLCs (Greater Boston Patient Care, etc.)
      │ keeps ~55% of G0156 rate as overhead
      ▼
HOME HEALTH AIDE
      receives ~$18.50/hr
      (sees ~40¢ of every taxpayer dollar)
```

## THE MATH — PER AIDE-HOUR

Using CTHRU home care appropriations for MVES FY2025:
- Community Choices: $64,220,131
- Home Care Services: $23,428,614
- **Total home care: $87,648,745**

```
$87.6M ÷ $40.72/hr = ~2,151,000 aide-hours billed to state

At $18.50/hr aide wage:
  Aide wages:        ~$39.8M
  Agency overhead:   ~$47.8M  (the spread)

Per hour billed:
  State pays:         $40.72
  Aide receives:     ~$18.50
  Agency keeps:      ~$22.22

AND this is AFTER Mystic Valley already took $33.3M
as the administrative middleman layer.
```
